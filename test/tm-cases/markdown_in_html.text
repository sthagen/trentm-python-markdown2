# Using headings to test toc and markdown-in-html together

- apples
- bananas

<div markdown="1">
This is *true* markdown text.
</div>

<div class="bar" markdown="1">
## This **too**.

And a paragraph
</div>

<div markdown="1" class="foo">
And **this**.
</div>

<section markdown='1'>
And even **this** in an HTML5 block tag.
</section>
